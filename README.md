# 📖 Resenhando

Aplicativo Android para registrar e organizar resenhas de forma simples e rápida.

<img src="https://github.com/walkerviani/assets/blob/main/pagina_inicial_resenhando.png" width="300"/>

## Sobre

O **Resenhando** é um app mobile que permite criar resenhas pessoais sobre livros, filmes, séries ou qualquer conteúdo que você queira registrar sua opinião.
